
== Data Asal 

Basis data dengan _schema_ PBB pada sistem basis data Oracle.

== Kegiatan Manipulasi Yang Dilakukan

Melakukan pengambilan data pembayaran PBB-P2 untuk tanggal 7 Oktober 2021 sebagai bahan rekonsiliasi data dengan Bank Jateng.

== Hasil Manipulasi

Hasil manipulasi data dapat diakses pada alamat https://docs.google.com/spreadsheets/d/19-hGYabumR_KTxxBhRrAHpHl2Ry7KCQn4kVIRaixl7Y/edit?usp=sharing

== _Script_ atau _command_ yang Digunakan Untuk Melakukan Manipulasi

----
select kd_propinsi||kd_dati2||kd_kecamatan||kd_kelurahan||kd_blok||no_urut||kd_jns_op nop, 
  thn_pajak_sppt, jml_sppt_yg_dibayar
from pembayaran_sppt
where tgl_pembayaran_sppt between to_date('071020210000','DDMMYYYYHH24MI') and to_date('081020210000','DDMMYYYYHH24MI')
order by kd_kecamatan, kd_kelurahan, kd_blok, no_urut, thn_pajak_sppt
----
